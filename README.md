1.  **Clone Repository**
    ```bash
    git clone https://github.com/kevinadisuryanugraha/clinic.0.git
    ```
2.  **Persiapan Server**
    - Karena website ini menggunakan PHP untuk fitur formulir, Anda memerlukan local server seperti **Laragon**, **XAMPP**, atau **MAMP**.
